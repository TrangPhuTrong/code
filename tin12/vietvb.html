Biện luận số nghiệm của phương trình bậc 2

Bài toán : Xác định số nghiệm của phương trình ax2 + bx - c = 0 (a != 0)

cách làm :

B1: Xác định hệ số a,b và c

B2: Tính delta = b2- 4ac

B3: Kết luận

+ Nếu delta < 0: phương trình vô nghiệm

+ Nếu delta = 0: phương trình có nghiệm duy nhất

+ Nếu delta > 0: phương trình có 2 nghiệm phân biệt
